# 🌿 Soothify – Online Therapy & Mental Wellness Platform

Soothify is a full-stack web application designed to provide accessible and personalized mental health support. The platform connects users with therapy services while helping them track their emotional well-being through interactive tools.

## 🚀 Features

- 🔐 User Authentication (Client Login System)
- 💳 Subscription Plans with Razorpay Integration (UPI & Cards)
- 📊 Personal Dashboard with Progress Tracking
- 📅 Appointment Booking & Session Scheduling
- 😊 Mood Tracker with Daily Insights
- 📓 Gratitude Journal & Personal Notes
- 🔔 Real-time Notifications & Reminders
- 📈 Progress Reports & Charts
- 🎯 Habit Tracker & Streak System
- 💬 Personalized Affirmations & Reflection Prompts

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: PHP
- Database: MySQL
- Payment Gateway: Razorpay

## 🎨 UI/UX

- Soft pastel blue themed design
- Fully responsive layout
- Smooth animations and user-friendly interface

## 📌 Purpose

The goal of Soothify is to make mental health support more engaging, structured, and accessible through technology.

## 🔮 Future Improvements

- Therapist Dashboard
- Admin Analytics Panel
- Video Call Integration
- AI-based Mood Insights
